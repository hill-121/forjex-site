工业流量计技术档案

覆盖液体、气体、蒸汽等多种介质的流量测量原理与参数参考

⚡

电磁流量计

热销

📐 工作原理

基于法拉第电磁感应定律，导电液体在磁场中流动时切割磁力线产生感应电动势，通过测量电动势计算流量。无活动部件，压损极小。

🎯 适用场景

自来水污水处理化工浆液食品饮料造纸冶金
口径范围DN10 ~ DN2000
精度等级±0.5%（可选 ±0.3%）
电导率要求≥ 5μS/cm
衬里材质橡胶 / PTFE / PFA / 陶瓷
输出信号4-20mA / 脉冲 / RS485 / HART
防护等级IP65 / IP68（潜水型）
🌀

涡轮流量计

📐 工作原理

流体推动涡轮叶片旋转，转速与流速成正比，通过磁电传感器检测叶片转速并换算为流量。响应快、精度高，适用于洁净低粘度液体。

🎯 适用场景

石油柴油天然气纯水溶剂液压油
口径范围DN4 ~ DN200
精度等级±0.5% / ±1.0%
量程比10:1 ~ 20:1
工作温度-20℃ ~ +120℃
输出信号脉冲 / 4-20mA / RS485
防爆等级ExdⅡBT4（可选）
🔊

超声波流量计

非接触

📐 工作原理

利用超声波在流体中顺流和逆流传播的时间差（时差法）或多普勒频移（多普勒法）来测量流速。外夹式安装无需断管，对流体无干扰。

🎯 适用场景

大口径管道自来水管网暖通空调冷却水原油非导电液体
口径范围DN15 ~ DN6000（外夹式）
精度等级±1.0%（时差法）/ ±2.0%（多普勒）
安装方式外夹式 / 插入式 / 管段式
流体温度-40℃ ~ +200℃（高温传感器）
输出信号4-20mA / 脉冲 / RS485 / Modbus
数据功能累积流量、热量计量（可选）
🌊

涡街流量计

📐 工作原理

在流体中放置阻流体（旋涡发生体），流体经过时交替产生卡门涡街，涡街频率与流速成正比。通过检测旋涡频率计算流量，适用于液体、气体和蒸汽。

🎯 适用场景

饱和蒸汽过热蒸汽压缩空气天然气氮气热水
口径范围DN15 ~ DN300
精度等级±1.0%（液体）/ ±1.5%（气体）
工作温度-40℃ ~ +350℃（高温型）
工作压力≤ 4.0MPa
输出信号4-20mA / 脉冲 / RS485 / HART
材质304 / 316L 不锈钢
⚖️

质量流量计（科里奥利）

高精度

📐 工作原理

基于科里奥利力原理，流体在振动测量管中流动时产生相位偏移，偏移量与质量流量成正比。可同时测量质量流量、密度和温度，精度最高可达 ±0.1%。

🎯 适用场景

化工配料石油贸易计量食品灌装制药高粘度介质
口径范围DN3 ~ DN150
质量流量精度±0.1% ~ ±0.2%
密度精度±0.002 g/cm³
量程比20:1
输出信号4-20mA / 脉冲 / RS485 / HART / Modbus
工作温度-50℃ ~ +200℃
🏞️

明渠流量计

📐 工作原理

通过超声波液位传感器测量明渠/堰槽中的液位高度，结合标准堰槽（巴歇尔槽、三角堰等）的水力特性曲线自动计算流量。非接触测量，安装简便。

🎯 适用场景

污水排放口自来水渠道灌溉渠道工业排水环保监测
测量范围0.01 ~ 9999 m³/h（依堰槽而定）
液位精度±2mm
液位量程0 ~ 2m / 0 ~ 5m / 0 ~ 10m
传感器类型超声波 / 雷达
输出信号4-20mA / RS485 / GPRS 远传
防护等级IP67
