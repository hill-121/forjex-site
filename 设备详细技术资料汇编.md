## 八、电磁流量计（仪器仪表线）

### 技术参数

| 参数 | 典型规格 |
|------|---------|
| 测量原理 | 法拉第电磁感应定律 |
| 口径范围 | DN10-DN2000 |
| 精度等级 | ±0.5% / ±0.3%（高精度型） |
| 流速范围 | 0.3-15 m/s |
| 电导率要求 | ≥5 μS/cm |
| 工作压力 | 0.6-4.0 MPa（高压型可达 100MPa） |
| 工作温度 | -20℃~180℃（取决于衬里材质） |
| 输出信号 | 4-20mA / 脉冲 / HART / RS485 |
| 供电 | AC 220V / DC 24V |
| 防护等级 | IP65 / IP68（潜水型） |
| 衬里材料 | 橡胶 / PTFE / PFA / 陶瓷 |
| 电极材料 | 316L / 哈氏合金 / 钛 / 钽 |

### 主流品牌

| 品牌 | 产地 | 价格区间（DN50） |
|------|------|-----------------|
| E+H（恩德斯豪斯） | 瑞士 | ¥8,000-15,000 |
| 横河（Yokogawa） | 日本 | ¥6,000-12,000 |
| 西门子 | 德国 | ¥7,000-13,000 |
| 科隆（KROHNE） | 德国 | ¥6,000-12,000 |
| 国产（川仪/开封/天信） | 中国 | ¥2,000-5,000 |

### 选型要点

1. **口径** — 根据管道流量计算
2. **衬里** — 橡胶（一般水）、PTFE（腐蚀介质）、陶瓷（磨损介质）
3. **电极** — 根据介质腐蚀性选择
4. **输出** — 4-20mA（简单）、HART（带通讯）、RS485 Modbus
5. **安装** — 一体式/分体式
